# あしあと機能 使用例

## 概要
「あしあとと里」画面では、ホーム画面でのつぶやきが自動的に「あしあと」として記録され、スキルの成長過程と感情の変化が視覚的に表示されます。

---

## 使用例1: 1日の感情の変化を追う

### ステップ1: 朝のつぶやき（ポジティブ）
**ホーム画面で入力:**
```
おはようございます！今日は新しいAPI設計書を作成する予定です。楽しみ！
```

**結果:**
- タスクボードに「API設計書の作成」が追加
- あしあとが保存される
  - スキル: システム設計
  - 感情: 😊（ポジティブ）
  - 色: 緑

---

### ステップ2: 午後のつぶやき（ネガティブ）
**ホーム画面で入力:**
```
データベース設計が思ったより難しい...調査を進めて解決策を見つけないと。
```

**結果:**
- タスクボードに「解決策の調査」が追加
- あしあとが保存される
  - スキル: 課題解決
  - 感情: 😰（ネガティブ）
  - 色: 赤

---

### ステップ3: 夕方のつぶやき（ポジティブ）
**ホーム画面で入力:**
```
やっと解決できた！先輩のアドバイスのおかげです。ありがとうございます！
```

**結果:**
- あしあとが保存される
  - スキル: コミュニケーション
  - 感情: 😊（ポジティブ）
  - 色: 緑

---

### ステップ4: あしあとを確認
1. 「🌲 あしあとと里」タブをクリック
2. プレイヤーから里への道に沿って3つの足跡が表示される
   - 1つ目: 緑の😊（朝のポジティブ）
   - 2つ目: 赤の😰（午後のネガティブ）
   - 3つ目: 緑の😊（夕方のポジティブ）
3. 足跡をクリックすると詳細が表示される

**表示例:**
```
😊 1/20
おはようございます！今日は新しいAPI設計書を作成する予定です。楽しみ！
スキル: システム設計
```

## あしあとの色の意味

| 色 | 感情 | 絵文字 | 例 |
|---|---|---|---|
| 🟢 緑 | ポジティブ | 😊 | 「嬉しい」「楽しい」「成功」「できた」 |
| 🔴 赤 | ネガティブ | 😰 | 「難しい」「困」「大変」「疲」 |
| 🟠 オレンジ | 通常 | 💭 | 上記以外の通常のつぶやき |

---

## スキルの自動判定

つぶやきの内容から自動的にスキルを判定します：

| スキル | キーワード例 |
|---|---|
| コミュニケーション | 会議、相談、話、コミュニケーション |
| システム設計 | 設計、アーキテクチャ、API |
| 課題解決 | 解決、調査、問題 |
| リーダーシップ | リーダー、チーム、マネジメント |
| 提案力 | 提案、アイデア、企画 |

---

## あしあとの確認方法

### 方法1: あしあとと里画面で確認
1. 「🌲 あしあとと里」タブをクリック
2. プレイヤーから里への道に足跡が表示される
3. 足跡をクリックすると詳細が表示される

### 方法2: 色で感情の変化を確認
- 緑が多い = ポジティブな期間
- 赤が多い = 困難な期間
- 色の変化 = 感情の波

### 方法3: 足跡の位置で成長を確認
- 里に近づくほど = スキルが成長
- 足跡の数 = 経験の蓄積

## Tips

### あしあとを効果的に使うコツ

1. **毎日つぶやく**
   - 朝、昼、夕方など、複数回つぶやくと感情の変化が分かりやすい

2. **正直な気持ちを書く**
   - ネガティブな感情も記録することで、乗り越えた軌跡が見える

3. **定期的に振り返る**
   - 週末に「あしあとと里」を見て、1週間を振り返る

4. **パターンを発見する**
   - 同じような感情の波が繰り返されていないか確認

5. **成長を実感する**
   - 過去の足跡を見て、自分の成長を実感する

## よくある質問

### Q: 足跡はいつまで保存されますか？
A: 最大20個まで保存されます。21個目以降は古いものから削除されます。

### Q: 足跡を削除できますか？
A: ブラウザのコンソールで `localStorage.removeItem('footprints')` を実行すると削除できます。

### Q: 足跡の色を変更できますか？
A: つぶやきの内容（ポジティブ/ネガティブ）によって自動的に色が決まります。

### Q: スキルの判定を変更できますか？
A: 現在は自動判定のみですが、将来的には手動で変更できるようにする予定です。

## まとめ

「あしあと」機能は、日々のつぶやきを視覚的な軌跡として記録し、スキルの成長と感情の変化を一目で確認できる機能です。

**主な利点:**
- 📊 成長の可視化
- 💭 感情の変化を客観視
- 🎯 スキルの偏りを発見
- 🌟 達成感の実感
- 🔄 パターンの発見

毎日つぶやいて、自分だけの「あしあと」を残しましょう！
